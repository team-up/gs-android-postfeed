**TeamUP Android**
=============================
토큰발급, 피드 게시 예제

**Config**
=============================
* TeamUP.CLIENT_ID

* TeamUP.CLIENT_SECRET

* TeamUP.ID `(사용자 계정)`

* TeamUP.PASSWORD `(계정 비밀번호)`

* TeamUP.FEEDGROUP_SEQUENCE `(피드그룹 번호)`

* TeamUP.TYPE_NORMAL `(피드 타입)`

* TeamUP.ALERT_OFF `(강제 알림 여부)`

**Library**
=============================
* Retrofit2
* LoganSquare
